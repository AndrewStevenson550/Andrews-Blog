## Getting Started

### Prerequisites

- Node.js (v18.x or later)
- npm
- PostgreSQL (or another database)

### Installation & Setup

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/Andrews-Blog.git
    ```
2.  Install backend dependencies:
    ```bash
    cd Andrews-Blog/server
    npm install
    ```
3.  Install frontend dependencies:
    ```bash
    cd ../client
    npm install
    ```
4.  Set up your environment variables by creating a `.env` file in the `server` directory. Use `.env.example` as a template.

5.  Run the application:
    ```bash
    # In the /server directory
    npm run dev
    ```
